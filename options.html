User Guide

Basic Usage

  1. Click extension icon on the page you want to test
  2. Use "Scan Forms" to detect forms on the page
  3. Enter required values in the form
  4. Save as profile with "Save Current Values"
  5. Use "Auto Fill" next time on the same page

Useful Features

  • Auto Fill: Automatically fill forms when page loads
  • Profile Management: Create and manage multiple profiles
  • Export/Import: Backup and restore data
  • Domain Exclusion: Disable features on specific sites

Important Notes

  • Avoid saving sensitive information (passwords, credit cards, etc.)
  • This extension is designed for test efficiency
  • Use with caution in production environments
  • Regularly backup your data

Troubleshooting

  • Forms not detected: Try rescanning after page fully loads
  • Auto-fill not working: Form structure may have changed
  • Data not saving: Browser storage limit may be reached
